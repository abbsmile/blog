from_unixtime(create_time)

发表于 2018-02-02 | 分类于 SQL

查看数据库数据，如果时间比如create_time是unix时间戳，必然很不方便。

可以使用from_unixtime函数进行格式转换。效果如下：

  • from_unixtime 有两个参数
    • 第一个参数可以是数据库字段，也可以是unix时间戳。
    • 第二个参数用于定义格式。可缺省。具体格式百度百科。
